Global Requirements
===================
1) development package of pidgin
2) osxcart (http://sourceforge.net/projects/osxcart/)

Compilation
===========
make

Installation
============
make install

Ubuntu Example
==============
sudo apt-get install pidgin-dev libpth-dev
and then install the osxcart from: http://sourceforge.net/projects/osxcart/
make
make install
